# ✈️ Flight Ticket Price Prediction

## 📌 Overview

A machine learning project that predicts flight ticket prices based on features like airline, route, class, duration, and days left before departure.

---

## 🧠 Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Streamlit

---

## 📊 Features Used

* Airline
* Source & Destination
* Departure & Arrival Time
* Stops
* Class
* Duration
* Days Left

---

## ⚙️ Workflow

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Model Training (Random Forest)
* Evaluation using R², RMSE

---

## 📈 Result

* Best Model: Random Forest
* R² Score: ~0.98

---

## 🖥️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
